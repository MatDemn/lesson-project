# MERN Lesson-Project

This application is about teaching and supporting students.


## Main features:

- Looking up current week's time table to make appointment easier
- Checking current week's your already booked lessons
- Exploring exercises with solutions


## Technical stack:

- MongoDB
- Express HTTP server
- React
- Node.js
- TypeScript

## Libs:

- Passport
- FontAwesome
- React-Bootstrap
- React Hook Form
- GoogleApis
- AWS-SDK
- Connet-Mongo
- Envalid
- Express-Session
- Http-Errors
- Mongoose

## Main technical features:

- AWS S3 bucket integration
- Google calendar integration
- Discord SSO login
